# Generador-de-contrase-as-con-Django
App para generar contraseñas en lenguaje Python y Framework Django

Instalar python: https://www.python.org/
Instalar los siguientes comandos para el gestor de paquetes de python pip:
MAC: curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python3 get-pip.py
Linux: sudo apt-get update
 sudo apt-get install python3-pip
 
Instalación del entorno virtual con virtualenv:
·	Dentro de la carpeta Django-Password-Generator, instalar el siguiente comando: pip3 install virtualenv.
·	Ejecutar el siguiente comando dentro de la carpeta Django-Password-Generator, para crear el entorno virtual: python3 –m virtualenv venv.
Para activar el entorno virtual ejecutamos los siguientes pasos:
·	En la carpeta Django-Password-Generator ejecutar cd venv para acceder a la carpeta del entorno virtual.
·	Ejecutar el comando cd bin.
·	Ejecutar el comando source activate para activar el entorno virtual.
Para instalar el framework Django:
En la terminal, dentro de la carpeta Django-Password-Generator, con el entorno virtual activado, se ejecuta el siguiente comando: pip install django.
Levantar el servidor.
·	Con el entorno virtual activado, y ubicandose en la raíz principal de la carpeta del proyecto Django-Password-Generator, ejecutar el siguiente comando para levantar el servidor: python3 manage.py runserver.
·	El servidor se levantará en el puerto: http://127.0.0.1:8000
